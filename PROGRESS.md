## 关键技术细节(便于新会话快速恢复上下文)
- **核心 bug 修复**:CSS transition 对 rotate 数值线性插值,wrap 到 0 会反向 360° 倒车
  - 解决:`currentMinutes` 不 wrap,无限累加,只在显示时 `% 720`
- **触摸交互**:只允许拖分针(教学聚焦);分针加 `::before` 隐形胖外套(~50px 宽触摸区)
- **模式切换**:body class `mode-sandbox` / `mode-challenge`,CSS 用 `.mode-*-only` + `!important` 控显隐
- **代码 vs 文案分离**:代码内部用 `sandbox`(class/变量),UI 文案用"自由玩 / 自由模式"(8 岁孩子能懂)
- **判定容差**:Step 4 用 wrappedDiff < 1.5 分钟(分针角度 ±3°)
- **指针动画**:平时 transition 0.15s 跟手;`.demo` 类切换到 1.6s 缓动用于自动演示
- **锁机制**:`isLocked` + body.locked,动画/演示期间禁拖动 + 禁提交
